hexxed
======

A portable, public-domain hex editor

![Hexxed with an open calculator dialog](https://i.imgur.com/66jwWmv.png)

Features:
* Multi-pane: Text and Hex pane (Code planned)
* Integrated 64-bit calculator
* Goto/Search
* Comments
* Bookmarks
* Highlighting and selection
